# Frontend

Aplicação Angular do Contract Predict. A página inicial apresenta o escopo acadêmico do projeto e
consulta a API para exibir seu estado de conexão.

## Executar

```bash
npm install
npm start
```

A aplicação estará disponível em `http://localhost:4200`.

## Rotas

- `/home`: apresentação institucional e status da API;
- `/info`: explicação técnica resumida do projeto;
- `/`: redireciona para `/home`.

## Build

```bash
npm run build
```
