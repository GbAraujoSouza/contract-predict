# Especificação inicial

## Objetivo

O Contract Predict investiga se sinais de microestrutura de mercado podem ajudar a prever se o
preço de um contrato binário do Polymarket irá subir nos próximos 15 minutos.

## Escopo desta fase

A primeira fase valida a arquitetura da aplicação:

- backend FastAPI executável;
- frontend Angular executável;
- consulta do frontend ao endpoint de saúde da API;
- páginas institucionais `/home` e `/info`;
- estrutura de diretórios para as próximas fases.

Não fazem parte desta fase: download do dataset, processamento de dados, treinamento de modelos,
persistência de resultados e dashboards.

## Roadmap

1. Base da aplicação.
2. Pipeline de dados e Machine Learning.
3. Dashboards com resultados reais.

## Aviso

O sistema é uma ferramenta acadêmica de análise e visualização. Ele não deve ser tratado como
recomendação financeira ou mecanismo para negociação real.
